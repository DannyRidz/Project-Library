# Library

A small browser-based library app built for The Odin Project JavaScript course.

## About

This project lets users add books to a personal library, display them on the page, mark books as read or not read, and remove books from the list.

The main goal of this exercise was to practice working with JavaScript objects, constructor functions, arrays, DOM manipulation, forms, and event listeners.

## Features

- Add a new book with a title, author, page count, and read status
- Display all books on the page
- Remove a book from the library
- Toggle a book between read and not read
- Hide and show the book form

## Built With

- HTML
- CSS
- JavaScript

## What I Learned

Through this project, I practiced:

- Creating objects with a constructor function
- Storing objects inside an array
- Using `crypto.randomUUID()` to give each book a unique ID
- Selecting elements from the DOM
- Creating new HTML elements with JavaScript
- Handling form submissions with `event.preventDefault()`
- Updating the page after the library array changes
